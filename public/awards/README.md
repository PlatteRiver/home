# Awards Logos

Place your award logo images in this folder.

## Required Award Logos:

1. **ESRI Partner Network Bronze**
   - Filename: `esri-partner-network-bronze.png` (or .jpg)
   - Recommended size: 300x200px or similar aspect ratio

2. **ArcGIS System Ready Specialty**
   - Filename: `arcgis-system-ready-specialty.png` (or .jpg)
   - Recommended size: 300x200px or similar aspect ratio

3. **ESRI Partner Conference 2024 Award Winner**
   - Filename: `esri-partner-conference-2024.png` (or .jpg)
   - Recommended size: 300x200px or similar aspect ratio

## Image Guidelines:
- Use PNG format with transparent backgrounds when possible
- Maintain consistent sizing across all award logos
- Ensure high resolution for crisp display
- Keep file sizes optimized for web (under 200KB each)
