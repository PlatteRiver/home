# Customer Logos

Place your customer/client logo images in this folder.

## Recommended Filenames:
- `customer1.png`
- `customer2.png`
- `customer3.png`
- `customer4.png`
- `customer5.png`
- `customer6.png`
- etc.

## Image Guidelines:
- Use PNG format with transparent backgrounds when possible
- Recommended size: 180x80px or similar aspect ratio
- Maintain consistent sizing across all customer logos
- Ensure high resolution for crisp display
- Keep file sizes optimized for web (under 100KB each)

## Note:
The page will automatically display placeholder text if an image is not found. Update the filenames in `src/pages/Home.jsx` to match your actual logo files.

You can add or remove customer logos by updating the array in the Customers section of `src/pages/Home.jsx`.
