# Partner Logos

Place your partner logo images in this folder.

## Recommended Filenames:
- `esri.png` - ESRI logo
- `partner2.png` - Additional partner logos
- `partner3.png`
- `partner4.png`
- etc.

## Image Guidelines:
- Use PNG format with transparent backgrounds when possible
- Recommended size: 200x100px or similar aspect ratio
- Maintain consistent sizing across all partner logos
- Ensure high resolution for crisp display
- Keep file sizes optimized for web (under 150KB each)

## Note:
The page will automatically display placeholder text if an image is not found. Update the filenames in `src/pages/Home.jsx` to match your actual logo files.
